<h1 align="center">✦ Portfolio Desain — Showcase Website</h1>

<p align="center">
Website ini dibuat sebagai tempat untuk menampilkan koleksi karya desain saya secara sederhana, rapi, dan mudah diakses. 
Fokus utamanya adalah memberikan pengalaman visual yang bersih sekaligus menunjukkan kemampuan saya dalam 
<strong>desain UI/UX</strong>, <strong>ilustrasi</strong>, dan <strong>proyek kreatif lainnya</strong>.
</p>

---

<h2>✦ Fitur Utama</h2>

<ul>
  <li><strong>Gallery Showcase</strong> — menampilkan karya desain dalam berbagai kategori</li>
  <li><strong>Halaman Arsip / Archive</strong> — berisi beberapa project yang sudah saya buat dan breakdown</li>
  <li><strong>Tampilan Responsif</strong> — optimal untuk desktop & mobile </li>
  <li><strong>Navigasi Sederhana</strong> — pengalaman browsing yang cepat dan jelas</li>
  <li><strong>Halaman Interaktif</strong> — dilengkapi dengan beberapa fitur interakti untuk meningkatkan kesan visual dan kenyamanan</li>
</ul>
